# Revenue — Company Revenue Dashboard

Aplicação fullstack para consulta, estimativa e análise de faturamento de empresas.

## Tecnologias
- Frontend: Next.js, React, Axios
- Backend: FastAPI, Supabase, Apollo API, OpenRouter
- Deploy: Vercel + Railway/Render/Docker

## Funcionalidades
- Cadastro e login
- Consulta de faturamento por empresa
- Busca em lote
- Controle de créditos
- Dashboard de uso
- Painel administrativo
- Exportação CSV
